# Sales Revenue Dashboard

## Project Overview
This project analyzes 1,000 sales records to track revenue, sales trends, and product performance. The dashboard uses data visualization and KPI monitoring to generate business insights and support data-driven decision-making.

## Dataset Features
- 1,000 Sales Records
- Product Categories
- Regional Sales Data
- Revenue Analysis
- KPI Tracking
